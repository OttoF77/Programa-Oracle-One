# 🚀 Oracle Next Education (ONE) - Exercícios

## 📋 Sobre

Este repositório contém todos os exercícios e projetos desenvolvidos durante minha jornada no Programa Oracle Next Education (ONE), uma iniciativa da Oracle em parceria com a Alura e Labora.

## 🎯 Programa ONE

O Oracle Next Education (ONE) é um programa de educação e empregabilidade com objetivo social de capacitar pessoas em tecnologia e conectá-las com o mercado de trabalho por meio de empresas parceiras.

## 📚 Trilhas

O programa é dividido em diferentes trilhas de aprendizado, abrangendo:

- Lógica de Programação
- Front-end
- Back-end
- Soft Skills & Empreendedorismo
- Inovação & Gestão

## 🛠️ Tecnologias Abordadas

- HTML & CSS
- JavaScript
- Java
- Spring Framework
- Git & GitHub
- Metodologias Ágeis
- E muito mais!

## ⭐ Desafios

Aqui você encontrará os desafios práticos propostos durante o programa, incluindo:

- Jogo do Número Secreto
- Jogo do Amigo Secreto
- E outros projetos empolgantes!

## 📝 Organização

Os exercícios estão organizados por módulos, facilitando a navegação e o acompanhamento do progresso.

## 🤝 Contribuições

Sinta-se à vontade para explorar, clonar e até mesmo sugerir melhorias através de pull requests!

## 📬 Contato
